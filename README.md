# TypeScript ハンズオンプロジェクト 🚀

TypeScriptの基礎を実践的に学べるハンズオンプロジェクトです。

## 🎯 このプロジェクトについて

5つのレッスンを通して、TypeScriptの基礎を段階的に学習できます：

1. **基本的な型** - プリミティブ型、配列、Enum、Union型
2. **関数** - 型付け、オプショナル引数、オーバーロード
3. **インターフェース** - オブジェクトの型定義と継承
4. **クラス** - OOP、継承、抽象クラス
5. **ジェネリクス** - 型の再利用とユーティリティ型

## 🚀 クイックスタート

```bash
# 依存関係をインストール
npm install

# レッスン1を実行
npm run lesson1

# レッスン2を実行
npm run lesson2

# その他のレッスンも同様に...
npm run lesson3
npm run lesson4
npm run lesson5
```

## 📂 プロジェクト構成

```
src/
├── 01-basic-types/      # Lesson 1: 基本的な型
├── 02-functions/        # Lesson 2: 関数
├── 03-interfaces/       # Lesson 3: インターフェース
├── 04-classes/          # Lesson 4: クラス
└── 05-generics/         # Lesson 5: ジェネリクス
```

各レッスンには`README.md`（解説）と`index.ts`（実践コード）があります。

## 💡 学習方法

1. 各レッスンのREADME.mdを読む
2. index.tsのコードを確認・編集
3. `npm run lesson*` で実行して動作確認
4. 練習問題に挑戦

Happy Learning! 🎉
